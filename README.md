# Django Blog

## О проекте

Данный репозиторий содержит тестовый блог-проект, разработанный на Django.
Визуальная составляющая построена с акцентом на современный внешний вид —
используются градиентные решения и анимации, реализованные средствами JavaScript.
Проект создавался как учебная площадка для отработки связки backend-логики
на Django с нетривиальным frontend-оформлением.

## Текущее состояние

Проект находится в стадии разработки и носит экспериментальный характер.
Основная цель на данном этапе — освоение принципов построения блог-платформы:
работа с моделями статей, система маршрутизации, шаблонизация, а также
интеграция динамических визуальных эффектов на стороне клиента.

Текущая кодовая база не претендует на production-качество — архитектурные
решения, структура приложения и визуальная часть будут пересмотрены
в следующих итерациях.

## Планы

Проект будет полностью переработан. Планируется смена визуальной концепции,
углублённая работа с анимациями, рефакторинг backend-части, добавление
системы аутентификации, комментариев, категорий и тегов. Текущая версия
является отправной точкой, на основе которой будет строиться полноценный
продукт.
